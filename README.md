# 올리브영 글로벌 GenAI 해커톤 Team 5

## 개요
- 국가별로 현재 활발하게 검색되는 키워드들이 모두 다르고, 빠르게 트렌드를 포착하는 과정에 드는 비용을 줄이기 위해 시작
- ex) 미국에서 비건이라는 키워드가 반복적으로 검색된다면, 해당 키워드를 통해 비건과 관련된 뷰티&헬스 케어 제품 기획전(프로모션)을 자동생성
- 검색 키워드 조사 -> 트렌드 파악 -> 매칭되는 상품 조사 -> 프로모션 기획 -> 프로모션 디자인 -> 프로모션 완성의 과정을 원클릭으로 해결 가능 예상

### 국가별 AI 기반 트렌드 키워드 프로모션 자동화
1. Google Trends의 실시간 트렌드 키워드 수집
2. 수집된 키워드를 통한 현상황 트렌드 파악
3. 트렌드와 올리브영 상품 데이터 매핑
4. 올리브영 상품 데이터 이미지를 활용한 프로모션 생성

### GenAI 기술 사용
> **사용 스택**

- Next.js (FullStack Framework)
- Supabase (Database)
- Gemini 2.5 pro (Thinking AI Model)
- Gemini 2.5 flash image (NanoBanana) (Image Generation)
- Vertex AI Studio (System Prompt Compare and Test)
- Vercel (Deploy)
